# MaxLights
A Windows app to control Lifx, WLED and RGB.Net LEDs.

# Installation
1) Download the latest release from https://github.com/stringandstickytape/MaxLights/releases .
2) Unzip it to the folder of your choice.
3) Run MaxLights.exe
